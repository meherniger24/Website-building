A fully parallel 3D thinning algorithm for giga-voxel scale microvasculature using OpenVDB

Description:

  • I implemented the Lohou parallel thinning algorithm in C++ to efficiently skeletonize large binary volumes; however, due to its reliance on 6-neighbor connectivity, it occasionally fails to fully reduce structures, especially in configurations requiring 26-neighborhood consideration. To address this, I integrated Lee et al.’s 1994 sequential thinning algorithm, which operates on full 26-neighbor adjacency, ensuring topological correctness and complete skeletal preservation. This hybrid pipeline—transitioning from parallel to sequential thinning—was then ported to OpenVDB, enabling scalable processing of gigavoxel-sized volumetric data with optimized memory usage and sparse data representation.
